## 🎯 BƯỚC TIẾP THEO (3 BƯỚC CUỐI)

### BƯỚC 1: Push lên GitHub (5 phút)

**File hướng dẫn:** `PUSH_TO_GITHUB.md`

```bash
# 1. Tạo repo trên https://github.com/new
# 2. Copy URL từ GitHub

# 3. Chạy lệnh này trong PowerShell:
git remote add origin https://github.com/YOUR_USERNAME/DoAnQuanLyPickleBall.git
git push -u origin main
```

### BƯỚC 2: Deploy Backend + Database lên Railway (15 phút)

**File hướng dẫn:** `DEPLOYMENT_GUIDE.md` - Phần "BƯỚC 2"

1. Vào https://railway.app
2. Login GitHub
3. New Project → Deploy from GitHub
4. Chọn repo
5. Thêm MySQL service
6. Setup Environment Variables
7. Import database

### BƯỚC 3: Deploy Frontend lên Vercel (10 phút)

**File hướng dẫn:** `DEPLOYMENT_GUIDE.md` - Phần "BƯỚC 3"

1. Build: `npm run build`
2. Vào https://vercel.com
3. New Project → GitHub repo
4. Framework: Vite, Root: frontend
5. Add env var: `VITE_API_URL=<Railway URL>`
6. Deploy
